# Contributing to OneScript Healthcare UI

Thank you for your interest in contributing to the OneScript Healthcare UI project!

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Start local server: `npm start`
4. Open `http://localhost:8080`

## Code Style

- Use consistent indentation (2 spaces)
- Follow existing code patterns
- Keep HTML semantic and accessible
- Use CSS variables for colors
- Comment complex JavaScript logic

## Adding New Pages

1. Create HTML file in appropriate folder (patient/, doctor/, etc.)
2. Include `js/app.js` script
3. Use shared components (header, footer)
4. Follow existing navigation patterns
5. Update README.md with new page

## Testing

- Test all navigation flows
- Verify forms work correctly
- Check responsive design on mobile
- Test in multiple browsers

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request with description

## Questions?

Open an issue on GitHub for any questions or suggestions.
